PIDinRootline

Prüft, ob angegebene Seite(n) innerhalb der Rootline sind.
Seit Typo3 v10 funktioniert [PIDinRootline = 34,36] bzw. [34 in tree.rootLineIds || 36 in tree.rootLineIds] nicht mehr in der User TSconfig.

// Beispiel: Default Upload-Ordner abhängig von der Seite anpassen. // Kommt in die `User TSconfig`-Datei options { defaultUploadFolder = 1:/bildarchiv/unsortiert/ } // Wenn der User im Backend auf der Seite / SysFolder 364 ist, dann anderen Ordner nehmen: [nnt3_pidInRootline(364)] options.defaultUploadFolder = 1:/uploads_kalender/ [GLOBAL] // Array von pids kann auch angegeben werden [nnt3_pidInRootline([364, 365])] options.defaultUploadFolder = 1:/uploads_kalender/ [GLOBAL]
